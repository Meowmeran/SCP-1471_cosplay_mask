# SCP-1471_cosplay_mask
Simplified and functional version of SCP-1471's face.

## Required components at the time:
- 1 x Wemos D1 Mini
- 2 x Neopixel WS2812B 4x4 RGB LED
  
- 1 x TP4056 Type-C
- 1 x MT3608 Boost Converter
- 1 x Power-Xtra PX663538 3.7V 1000 mAh Li-Polymer Battery


- 820uF 25V Eletrolitic Capacitor
- TTP223B Buttons
- Resistors
- Wires
